- 这是一项通过表征相似性分析，分析语言模型神经网络语言表征与人脑语言表征的相似性的工作
- EEG数据体积超过1GB，data文件夹没有上传
- 结论如下：
1. 和人脑越相似的层，对模型表现越重要
2. 不同模型对不同的EEG时间窗口呈现不同的相似性
3. 越深的层，和人脑越相似
